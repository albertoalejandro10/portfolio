## Descripción General

El Sistema de Gestión de Inventario es una aplicación web con todas las funciones construida para un negocio local para gestionar eficientemente sus operaciones de inventario. El sistema proporciona una solución completa para rastrear productos, gestionar niveles de stock, procesar ventas y generar informes detallados. Construido con PHP 7 y utilizando la plantilla AdminLTE, ofrece una interfaz profesional y fácil de usar para las operaciones comerciales diarias.
